# GPT Image 2 — 30 Prompts E-commerce & Email Marketing

Guia interativo com 30 prompts profissionais focados em e-commerce e email marketing para o GPT Image 2 da OpenAI.

## 🚀 Como publicar na Vercel

1. **Suba esta pasta inteira pro GitHub** (mantendo todos os arquivos)
2. **Acesse [vercel.com](https://vercel.com)** e faça login com o GitHub
3. **Clique em "Add New Project"** e selecione este repositório
4. **A Vercel detecta automaticamente** que é um projeto Vite — só clique em **Deploy**
5. Pronto, em ~1 minuto seu site está no ar

## 💻 Rodar localmente (opcional)

Se quiser testar antes de publicar:

```bash
npm install
npm run dev
```

Abre em `http://localhost:5173`

## 📁 Estrutura

- `src/App.jsx` — componente principal com os 30 prompts
- `src/main.jsx` — entrada do React
- `index.html` — página HTML base
- `vite.config.js` — configuração do Vite
- `vercel.json` — configuração de deploy
- `package.json` — dependências do projeto
